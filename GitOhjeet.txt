Working dir -> git add (esim tiedosto) -> git commit ->
Commit --> asetetaan joku tiedosto seurantaan ---> tekee arkistosta (repo) sen hetkisen "snapshotin" 
Kirjaa viesti 
	git commit -m "my message (esim version vaihe)"
Commit messaget kannattaa tehdä alusta lähtien selkeästi
